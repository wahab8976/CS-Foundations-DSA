# Undergraduate topics in Computer Science - for Real Scientists

Welcome to my personal archive of Data Structures, Algorithms, and Competitive Programming templates! 

This repository is a curated collection of everything I have conceptualized, solved, and optimized throughout my undergraduate journey. It serves as both my personal contest cheat sheet and a testament to my growth in algorithmic problem-solving.

I am currently in my final year of graduation, driven by a deep interest in engineering elegant solutions to complex algorithmic challenges. I am building and maintaining this repository as a commitment to continuous learning—aiming to become a better version of myself as a software engineer every single day.

## What's Inside?
* **Topic-Wise Implementations:** Clean, modular, and highly optimized templates for Graphs, Mathematics, Trees, and Dynamic Programming and many other topics.
* **Competitive Programming Hacks:** Speed optimizations, compiler built-ins, and standard boilerplate setups to avoid TLEs.
* **Interview-Ready Code:** Implementations written with industry-standard code readability and detailed explanations.

## Goals of this Repo
1. Avoid reinventing the wheel during timed contests (Codeforces, LeetCode, CodeChef).
2. Deepen my understanding of space-time complexities through hands-on implementation.
3. Build a reliable, production-grade reference manual for future engineering interviews.
